BOM for shift knob PCB:

- power regulation for using 12V input from car
- Arduino Mini (perhaps the nano if I want integrated USB) 
- usb interface (for connecting to the Mini)
- motor controller for 12V supply 
  - average desired output = 5V
  - peak current = 500 mA
- 7 segment
- shift register(s) for 7 segment
- button(s)
- 3-5 external LEDs
- resistors
- capacitors
- linear power supply
- accelerometer (optional)
- connection to external accelerometer (optional)
- bluetooth interface (optional)
